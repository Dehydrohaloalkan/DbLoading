# 14. Глоссарий

- **DB2**: база данных IBM DB2 (внешняя зависимость).
- **Group (Группа)**: логическая вкладка/раздел UI, объединяет логические скрипты.
- **Script (Логический скрипт)**: единица выбора/запуска пользователем. Состоит из 3 вариантов (SQL файлов).
- **Variant (Вариант)**: один `.sql` файл, возвращающий строки результата. Выполняется как часть Script.
- **Run**: один запуск выгрузки пользователем, имеет `runId`, статусы и результаты.
- **Execution lane**: “поток” выполнения SQL. Внутри lane задачи выполняются последовательно.
- **LineFile**: колонка результата в default режиме (`SELECT "LineFile"`), готовая строка.
- **Custom columns**: режим, где `SELECT "LineFile"` заменяется на выражение склейки выбранных колонок/выражений через `|`.
- **Slicer**: компонент, который нарезает поток строк на файлы фиксированного максимального размера (10 MiB).
- **Stream**: внешний получатель файлов (будущее расширение); писать в stream надо после успешной выгрузки на диск.
- **Observer**: паттерн “наблюдатель” — подписчики на события запуска/завершения выгрузок.
